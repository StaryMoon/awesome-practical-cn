# 开源替代软件清单

> 分类：**开源 / 替代**
>
> 适合：想找免费、开源、跨平台软件的人
>
> 截图来源：[https://github.com/obsproject/obs-studio](https://github.com/obsproject/obs-studio)

## 一句话

整理常见商业软件的开源替代入口，以及图片、视频、音频、办公和直播工具。

## 为什么值得收藏

开源替代是 GitHub 上非常吃香的主题，搜索意图明确，用户愿意 star 保存。

## 快速上手

1. 先列出你正在用的付费工具。
2. 逐个找开源替代并试用一周。
3. 确认文件格式兼容后再迁移。

## 常见坑

- 开源不等于免费商用无条件，仍要看 license。
- 不要为了开源而牺牲关键工作流稳定性。

## 维护建议

- 如果某个工具出现价格、额度、开源状态或官网迁移，请优先改本页链接和说明。
- 如果补图，请使用官方公开页面截图，并保留来源链接。
- 如果新增入口，请写清楚它解决什么问题，避免变成无差别链接农场。
