## Overview

整理密码管理、加密通信、隐私邮箱、浏览器、安全检查和数据泄露查询入口。

## Scope

本页只收录与该主题直接相关、入口稳定、说明清晰的资源。优先选择官方文档、主流开源仓库、长期可访问的产品页面和常用工具链。

## Recommended Path

1. 先启用密码管理器和两步验证。
2. 重要账号使用独立邮箱和独立密码。
3. 定期查泄露并轮换密码。

## Notes

- 避免复用密码。
- 2FA 恢复码不应与账号密码存放在同一云笔记中。
