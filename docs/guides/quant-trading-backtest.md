## Overview

整理开源交易机器人、回测框架、行情接口和 A 股/加密数据工具，强调风控和纸交易。

## Scope

本页只收录与该主题直接相关、入口稳定、说明清晰的资源。优先选择官方文档、主流开源仓库、长期可访问的产品页面和常用工具链。

## Recommended Path

1. 只做纸交易，先跑 3 个月模拟。
2. 每个策略必须有最大回撤、手续费和滑点。
3. 用小资金之前先写风控规则。

## Notes

- 截图收益率不能作为策略有效性的证据。
- 回测盈利不等于实盘盈利，本页不构成投资建议。
