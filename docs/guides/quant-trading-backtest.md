# 量化交易、回测与行情数据

![量化交易、回测与行情数据](../../assets/screenshots/quant-trading-backtest.png)

> 分类：**金融 / 量化**
>
> 适合：想做量化机器人、回测、纸交易和数据分析的人
>
> 截图来源：[https://github.com/freqtrade/freqtrade](https://github.com/freqtrade/freqtrade)

## 一句话

整理开源交易机器人、回测框架、行情接口和 A 股/加密数据工具，强调风控和纸交易。

## 为什么值得收藏

量化赚钱叙事很容易吸引关注，但高质量项目必须把风险和回测讲清楚。

## 快速上手

1. 只做纸交易，先跑 3 个月模拟。
2. 每个策略必须有最大回撤、手续费和滑点。
3. 用小资金之前先写风控规则。

## 常见坑

- 不要相信截图收益率。
- 回测盈利不等于实盘盈利，本页不构成投资建议。
