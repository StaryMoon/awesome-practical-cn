> Category: **AI / API**
>
> Audience: 做 AI 应用、机器人、Agent、论文实验的人
>
> Screenshot: [https://openrouter.ai/](https://openrouter.ai/)

## Overview

整理常用大模型 API 平台、聚合平台和模型托管入口，方便快速比较模型、文档、控制台和定价入口。

## Scope

本页只收录与该主题直接相关、入口稳定、说明清晰的资源。优先选择官方文档、主流开源仓库、长期可访问的产品页面和常用工具链。

## Recommended Path

1. 先确定模型用途：文本、代码、视觉、语音或 embedding。
2. 把控制台、定价页、API 文档一起收藏。
3. 先写最小 demo，再接入生产。

## Notes

- 优先确认地区、支付方式、速率限制和账单策略。
- 生产场景需评估聚合平台的稳定性、隐私边界和服务可用性。

## Maintenance

- Update links when official pages, pricing, quotas, or open-source status change.
- Use screenshots from public official pages and keep the source URL.
- Describe the concrete use case for each new entry.
